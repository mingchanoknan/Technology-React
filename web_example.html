บริษัทระดับโลกที่ใช้ "React" พัฒนาเว็บ

ปัจจุบันบริษัท Tech มากมายใช้ React เพื่อพัฒนาเว็บไซต์ของตัวเอง เราจะมาดูกันว่าบริษัทไหนบ้างที่ใช้ React และอะไรที่ทำให้บริษัทยักษ์ใหญ่นี้เลือกที่จะใช้ React กัน

Facebook

  บริษัทแรกที่เราจะไม่พูดถึงไม่ได้เลยคือ Facebook ผู้คิดค้นและสร้าง React ให้เราใช้งานกันอยู่ทุกวันนี้ โดยที่เว็บของ Facebook นั้นสามารถ Interact กับ User ได้โดยทันทีไม่จำเป็นต้อง Reload ทั้ง Page นอกจากนี้ App Facebook และ Instagram ก็ถูกสร้างด้วย React Native เช่นกัน

Atlassian

  ผู้เป็นเจ้าตลาดเครื่องมือในการบริหารงานเพิ่ม Productivity ยอดนิยม เช่น JIRA, Confluence, Bitbucket, Trello, Hipchat ซึ่งแน่นอนว่า App เหล่านี้ก็ถูกสร้างด้วย React ด้วยเช่นกัน โดยที่ Atlassian เลือกใช้ React ในการพัฒนาเพราะ React นั้นสามารถพัฒนาได้ง่าย แถมยังรองรับได้หลายอุปกรณ์ เช่น ไม่ว่าจะเป็นบนโทรศัพท์ หรือคอมพิวเตอร์ก็ใช้งานได้

UberEats

  แอปพลิเคชันสั่งอาหารเจ้าแรกๆ ของโลกที่ได้รับความนิยมอย่างมากในอเมริกา ได้เลือกใช้ React เพื่อพัฒนาเว็บไซต์สำหรับการหาร้านอาหาร, สั่งอาหาร รวมถึงการ Track Order ต่างๆ ส่วนเหตุผลที่ UberEats เลือกใช้ React ก็หนีไม่พ้นเรื่องการรองรับการใช้งานได้บน Smartphones และ Tablets ซึ่งทำให้ลูกค้า ร้านอาหาร และไรเดอร์ใช้งานได้อย่างสะดวก และมีประสบการณ์ที่ดีในการใช้งานนั่นเอง

UberEats

Netflix

  เจ้าตลาด Video Streaming เลือกใช้ React ในการพัฒนาเว็บไซต์เพื่อ Redesign User Interface ตั้งแต่ปี 2015 โดยสาเหตุที่ Netflix ตัดสินใจเปลี่ยนมาใช้ React เนื่องจากประสิทธิภาพของเว็บที่ได้, ความเร็วในการปล่อย Software, การสามารถแบ่งเป็น Module ย่อยๆในการพัฒนา โดยยังสามารถมี Feature รองรับการเล่นวีดิโอแบบ 4K, HDR และ Dolby Visio

Netflix

Dropbox

  เว็บฝากไฟล์เจ้าแรกบนโลกที่ก่อตั้ง และดำเนินกิจการมาตั้งแต่ปี 2012 แต่ก็เจอเข้ากับปัญหาในการพัฒนาเว็บไซต์ จนกระทั่งในปี 2016 Dropbox เลือกที่จะปรับ Design Library โดยใช้ React ทำ Functions ต่างๆ เพื่อรองรับการทำงาน เช่น การเก็บรวบรวมและการคำนวณขนาดไฟล์ต่างๆ นอกจากนี้ยังมี Features ในการ Show Page ตอนที่ Scroll Down ต่างๆ นอกจากนี้ทีมงาน Dropbox ยังช่วยพัฒนา React และ JS Tools อื่นๆ เพื่อให้ทำงานได้ดีขึ้นบน Web Browser ต่างๆ
